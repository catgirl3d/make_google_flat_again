# Changelog

## v0.1.2 - 2026-06-19

This release makes the return of classic Google Workspace icons more complete and more reliable.

### Key Changes

- Classic tab icons have been added for Docs, Sheets, Slides, Forms, and Vids.
- Classic header branding now works better in Google Forms and Google Vids.
- Fixed cases where the new Google favicons could reappear after the page finished loading.
- Improved extension stability in Chrome.
